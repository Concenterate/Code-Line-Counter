# Code Line Counter
Python script for iterating directory and outputting total line counts for all file types.

## Useage Example
`python code-line-counter.py "C:\WORK\Some-Project"`

Example Output:
FILE TYPE  | LINE COUNT
.cs        | 3455
.cshtml    | 3077
.csv       | 1779
.py        | 33
.xml       | 84242
.css       | 31321
.html      | 27195
.exe       | 58
.php       | 313
.json      | 392
.c         | 701
TOTAL      | 559230
